# Project 3: Spots

### Description

Spots is an image sharing site that adapts to moblie screens up to desktop screens. Ability to make new post and edit profile, while liking photos.

**Technologies and Techniques Used**

- HTML
- CSS
- BEM
- Flexbox & Grid Layout
- Responsive design
